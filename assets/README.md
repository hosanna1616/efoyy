# Assets Directory

Place your assets here:

- `animations/` - Rive animation files (.riv)
- `maps/` - Hospital floor plan images or map tiles
- `images/` - App icons, logos, patient photos
- `lottie/` - Lottie animation files (.json)

## Optional Assets

For a complete experience, you can add:

1. **Lottie animations** for:
   - Queue waiting animation
   - Success/confirmation animations
   - Loading states

2. **Hospital floor plans** as images or map tiles for navigation

3. **App icon** and splash screen images
