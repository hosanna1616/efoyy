# App Icon Creation Instructions

Since we can't generate images directly, please create an app icon with:
- Size: 512x512 pixels
- Background: Green (#4CAF50) - representing health/medical
- Foreground: White medical cross or "E" for Efoy
- Format: PNG

Or use an online tool like:
- https://www.appicon.co/
- https://icon.kitchen/

Place the icon as: `android/app/src/main/res/mipmap-xxxhdpi/ic_launcher.png`
